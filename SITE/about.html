О проекте

Проект разрабатывается студентами Московского Политеха для решения проблемы нехватки специалистов по БПЛА.

Проблематика

  • Высокая стоимость реальных квадрокоптеров для обучения
  • Недостаток практических тренажеров на рынке
  • Дефицит специалистов по БПЛА

Цели и задачи

  1. Создание 3D моделей компонентов квадрокоптера
  2. Разработка VR-интерфейса управления
  3. Программирование логики приложения
  4. Портирование на мобильные платформы
