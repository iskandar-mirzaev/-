О проекте

Наш проект представляет собой VR-симулятор для обучения сборке электрических схем квадрокоптеров, разрабатываемый студентами Московского Политеха.

Актуальность проекта

В связи с ростом популярности БПЛА возникает острая необходимость в квалифицированных специалистах. Наш симулятор решает проблему дороговизны реальных тренажеров.

Проблематика

  • Высокая стоимость реальных квадрокоптеров для обучения
  • Недостаток практических тренажеров на рынке
  • Дефицит специалистов по БПЛА

Цели и задачи

Основная цель

Создание доступного VR-тренажера для подготовки специалистов по сборке БПЛА

Задачи проекта:

  1. Разработка 3D-моделей компонентов
  2. Создание VR-интерфейса
  3. Программирование логики сборки
  4. Тестирование и отладка
  5. Портирование на мобильные платформы
