О проекте

Проект разрабатывается студентами Московского Политеха для решения проблемы нехватки специалистов по БПЛА.

Проблематика

  • Высокая стоимость реальных квадрокоптеров
  • Недостаток практических тренажеров

Задачи проекта

  1. Создание 3D моделей компонентов
  2. Разработка VR-управления
  3. Программирование логики приложения
  4. Портирование на Android
